3 - vamos ir na model


class Schedule extends Model
{
    /** @use HasFactory<\Database\Factories\ScheduleFactory> */
    use HasFactory;

    protected $casts = [
        'started_at' => 'date',
        'ends_at' => 'date',
    ];

    public function employee()
    {
        return $this->belongsTo(Employee::class);
    }
}

<!-- No Laravel, a propriedade $casts dentro de um model é usada para converter automaticamente os atributos do banco de dados para tipos específicos ao acessá-los. 


Isso significa que os atributos started_at e ends_at serão automaticamente convertidos para instâncias de Carbon (a biblioteca de manipulação de datas do Laravel).

Benefícios:
Manipulação de Datas
Você pode usar métodos do Carbon diretamente nesses atributos:
